# 國際化（i18n）

採用 Django 內建 i18n，支援 **繁體中文** 與 **English** 切換。

## 語言切換

- 導覽列右上角（桌面）或手機選單內有語言下拉選單
- 首頁（`feed`）示範中英文文案切換
- URL：`POST /i18n/setlang/`

## 設定

`mysite/settings.py`：

- `LANGUAGE_CODE = 'zh-hant'`
- `TIME_ZONE = 'Asia/Taipei'`
- `LANGUAGES`：zh-hant、en
- `LOCALE_PATHS`：`locale/`

## 翻譯檔

```
locale/
  zh_Hant/LC_MESSAGES/django.po
  en/LC_MESSAGES/django.mo
```

## 編譯翻譯（Windows）

若無 GNU gettext，請執行：

```bash
pip install polib
python scripts/compile_locale.py
```

Linux 正式環境可使用：

```bash
python manage.py compilemessages
```
